UI minimalis untuk hackathon yang cepat dipakai dan mudah dipresentasikan.

Frontend ini pure HTML + CSS. Bagian demo menampilkan contoh response API (statis) supaya gampang dipahami tanpa butuh framework atau setup rumit.

Cara Menjalankan

Frontend

Buka file frontend/index.html langsung di browser (tanpa server).

Backend

Jalankan: mvn -f backend/pom.xml spring-boot:run

Cek Endpoint

Test: curl -s http://localhost:8080/api/v1/health
